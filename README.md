# AI Career Agent

An intelligent automation tool that reads job profiles, tailors your resume to match job requirements, and automates the job application process.

## Features

- Automated job profile analysis and parsing
- Resume matching and customization based on job requirements
- Intelligent skill alignment with job postings
- Automated job application submission
- Resume optimization for Applicant Tracking Systems (ATS)
- Job requirement gap analysis

## Installation

### Prerequisites
- Python 3.8+ or Node.js 14+
- Git

### Setup

1. Clone the repository:
```bash
git clone <repository-url>
cd ai_career_agent
```

2. Install dependencies:
```bash
# For Python projects
pip install -r requirements.txt

# For Node.js projects
npm install
```

## Usage

```bash
# Start the application
npm start
# or for Python:
python main.py
```

## Configuration

Create a `.env` file in the root directory with the following variables:
```
API_KEY=your_api_key_here
ENV=development
```
